📰 Portal de Notícias Tech

Uma aplicação web moderna desenvolvida do zero com foco em experiência do usuário, organização de conteúdo e interface inspirada em grandes portais de notícias de tecnologia.

O projeto apresenta notícias organizadas por categorias, destaques visuais e uma navegação limpa para proporcionar uma leitura intuitiva e agradável.

🧠 Conceito do Projeto

O objetivo deste projeto foi criar um portal de notícias com identidade visual moderna e foco em conteúdo tecnológico.

A aplicação foi construída inteiramente por mim, desde a estrutura da interface até a organização dos componentes visuais, buscando praticar conceitos importantes de desenvolvimento front-end como:

Estruturação semântica
Organização visual
Responsividade
Componentização de layout
Experiência do usuário (UX)
Boas práticas de desenvolvimento
